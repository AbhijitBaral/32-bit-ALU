# 32 bit ALU  

Verilog description of 32 bit ALU operating on integer type data.  

Supported operations:  
 * Addition/Subtraction
 * Multiplication
 * Division
 * Increment/Decrement by 1
 * 2's complement
 * Bitwise AND, OR , EXOR
 * Arithmetic Shift( left, right)
 * Logical shift (left, right)
 * Rotate
